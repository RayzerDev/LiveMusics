A propos

1. Auteurs du site

Le site a été conçu en groupe de 3. Il possède donc 3 propriétaires, des etudiants du BUT Informatique de l'IUT de Lens :

  1. Louis KARAMUCKI
  2. Raphael DELPIRE
  3. Théo VAASTT

Ce site a pour but d'être évaluer pour la SAE 1.5 et 1.6 de ce même BUT.

2. Accessibilité

Nous avons rendu le site responsive, avec le CSS en utilisant les propriétés de l'écran pour s'adapter à l'appareil. Il est donc adapté à tout les appareils.

3. Conception

Ce site a été pensé avec Figma pour réaliser les maquettes. Le site utilise HMTL/CSS uniquement. La page d'accueil est accessible partout, en appuyant sur le logo à gauche. Le site contient 7 pages :

  1. L'accueil
  2. Les abonnements
  3. L'agenda
  4. A propos
  5. Candidature
  6. Contact
  7. Studios

Elles sont toutes accessible entre elles.
